Detailed Description

Complements TyphoonAssistedFactoryBase to allow methods defined by closures.

The implementation overrides methodSignatureForSelector and forwardInvocation to implement the methods defined by the closures.

Even if this class is "internal", the prefixes avoid the possibility of the user defining some factory method with the same name.

Users should not use this class directly.
